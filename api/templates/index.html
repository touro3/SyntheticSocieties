⚡ Quick Start

# Check the API is live
curl {{ base_url }}/health

# List all experiments
curl {{ base_url }}/experiments

# Trigger a simulation (CPU policies: random, rule_based, mock)
curl -X POST {{ base_url }}/simulate \
  -H "Content-Type: application/json" \
  -d '{"config_path": "configs/base_config.yaml"}'
# Clone and run with Docker Compose
git clone https://github.com/touro3/SyntheticSocieties.git
cd SyntheticSocieties
docker compose up --build

# API is live at http://localhost:5050
curl http://localhost:5050/health

# Run a simulation (results persist to ./experiments/)
curl -X POST http://localhost:5050/simulate \
  -H "Content-Type: application/json" \
  -d '{"config_path": "configs/base_config.yaml"}'
# Clone and install
git clone https://github.com/touro3/SyntheticSocieties.git
cd SyntheticSocieties
pip install -r requirements.txt   # full deps (includes torch)
pip install -e .

# Run the full pipeline (CPU baselines, no GPU needed)
python scripts/run_full_pipeline.py --rounds 10 --agents 20 --seeds 42

# Or start the API server locally
python api/app.py

# With GPU (LLM policies)
python scripts/run_full_pipeline.py --include-llm --rounds 30 --agents 100

Architecture

☁️ Cloud (Render Free Tier)

API deployed on Render.com — zero cost, auto-deploys from GitHub.

  • CPU-only policies (random, rule_based, template, mock)
  • 512 MB RAM — lightweight, no GPU deps
  • Spins down after 15 min idle (cold start ~30s)
  • Runs real simulations with non-LLM policies

💻 Local (Docker / Bare Metal)

Full simulation engine on your machine — GPU support for LLM policies.

  • All policies including LLM (Mistral-7B, GPT, etc.)
  • GPU acceleration via CUDA / bitsandbytes 4-bit
  • Persistent experiment data in ./experiments/
  • docker compose up or pip install + python api/app.py
